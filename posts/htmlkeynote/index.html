HTML常用标签

一. a

属性:

  1. href
    1. 绝对路径
      • <a href="/a.com">_</a>
    2. 相对路径
      • <a href="./a.com"></a> or <a href="./a.com">_</a>
    3. 伪协议　只执行代码逻辑,不进行跳转
      • <a href="javascript:code;">_</a>
      • <a href="mailto:email;">_</a>
      • <a href="tel:phonecall;">_</a>
  2. download // 下载相关
  3. target
    1. _blank 新页面打开
    2. _self 在自身内打开（默认值）
    3. _top 嵌套iframe src 在最外层打开 name定义窗口门
    4. _parent 嵌套iframe在外层打开
    5. 自定义名字跳转到自定义名字窗口 target='xxx’,给新打开的窗口window.name='xxx’

二. 表格相关

标签: table thead tbody tfooter tr th tr td

  1. table > thead tbody tfooter
  2. thead tbody tfooter > tr
  3. tr > th td
  4. 可表格嵌套

table 常用属性

  1. table-layout: auto | fixed;
    • 规定单元格大小的情况下
    • auto 单元格宽度自动根据内容大小分配
    • fixed 单元格宽度尽量平均分配
  2. border-collapse: number 边框合并
  3. border-spacing: number 边框距离

img 图片

  1. src 资源地址
    • <img src="相对路径/绝对路径/ //">
  2. width
    • <img src="相对路径/绝对路径/ //" width="120">
  3. height
    • <img src="相对路径/绝对路径/ //" height="120">
  4. alt 图片裂了代替展示
    • <img src="相对路径/绝对路径/ //" alt="裂开了">
  5. onload onerror 事件
    • 可用js在监听onload和onerror事件在发生事件后作出响应的动作

form input 都要有name

  1. action 表单提交地址
  2. method 提交方式get or post
  3. autocomplete on/off 是否自动补全
  4. target 提交到其他页面
  5. onsubmit 用户提交事件 内部要有提交按钮
